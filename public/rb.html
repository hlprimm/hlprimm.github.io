Civcraft transport map's RealisticBiomes configuration spreadsheet

All values are pulled from config file in official repo. Percentages are calculated on page load, based on interpretation of the plugin's code. This interpretation may be wrong, contact u/the_gipsy if you find errors.

Notes:

  1. Hover over table headers to get more information
  2. "Ideal conditions" column assumes full sunlight, except for netherwards as noted
  3. Some biomes are correctly conflated for readability (e.g. "ICE"), reflecting configuration by 06-12-2014. A future configuration change might break current conflation.
  4. AFAIK sunlight level means exposure to sky: transparent blocks are taken into account and time of day is irrelevant.
  5. I am not 100% positive about horse stats being normalized and not inherited.
  6. It is unclear why chicken spawn rate is not limited, despite that there exists code for it. My guess is that the Bukkit API broke at some point for that event.
  7. You should be able to figure everything out from this table. For a general overview about RealisticBiomes please check the wiki.
